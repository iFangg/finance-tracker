# finance-tracker

Use this finance tracker to log transactions on spreadsheets. Start the finance tracker via the command

```python
python3 finance_tracker.py
```

You will be prompted with 4 inputs one after the other:

```python
BUSINESS: 

DESCRIPTION:

AMOUNT (e.g 100 | 10.00 | 29.35 | -99.21):

DATE (01/01/2001 | 1/1/2001 | Leave blank for today's date):
```

The first three inputs will not accept null or empty inputs and will continue to prompt you until you provide valid input.
